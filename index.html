Company Portal App

The Company Portal App provides a way to organize or have a structured view of a company's workforce. There are two types of accounts: User and Admin. Users are limited to seeing what announcements the company gets and projects they are assigned to. Admin accounts have access to all company and team info for all users.

  • Technologies Used: Figma, HTML, CSS, TypeScript, Java, PostgreSQL, Angular, Postman

Spotify Who's Who

This is a game that uses the Spotify API to source the question of Who the mystery singer is. The user is able to set the amount of song samples received and the number of multiple choices to choose from. They can also pick from any genre recognized by Spotify.

  • Environment: Front End
  • Technologies Used: Figma, HTML, SASS, TypeScript, Angular

Bamazon

Bamazon is a Command Line Interface powered storefront with customer and supplier functionalities. Currently, the two modes available are customer and stock supplier. In customer mode, the user is able to make purchases of what is in stock. In stock supplier mode, the user has options to add items, remove items, or replenish stock. This application models any store or manufacturing environment. Future Scope for this project would include a supervisor or Administrator mode.

  • Environment: Command Line Interface (CLI)
  • Technologies Used: MySQL, JavaScript, NodeJS

Drink Forecast

A fun app that makes beverage suggestions to a user based on their location and the current temperature at that location. App uses Geolocation to find users automatically. Also, OpenWeather API provides temperature data for locations, while the BeverageAPI provides list of beverages.

  • Environment: Front End
  • Technologies Used: HTML, CSS, JavaScript
